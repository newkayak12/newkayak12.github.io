# package, import

## 구조
- `package`로 시작하고 `.`로 붙는다. java와 같다.

## import 사용하기
- import directive를 사용하면 전체 이름을 사용하지 않아도 된다.
- `import java.lang.Math` // JDK 클래스
- `import foo.bar.util.readInt` //최상위 함수

> 특이 사항
> - kotlin은 static import가 없다.
> - 그냥 import로 사용하면 된다.
