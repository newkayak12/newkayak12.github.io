## Netstat

```linux
netstat [option] [|grep portNumber or ServiceName]

```
## options
- `-l` (listen) : 연결 가능한 상태
- `-n` (number port) : port
- `-t` (tcp) : tcp
- `-u` (udp) : udp
- `-p` (pid) : PID
- `-a` (all) : all
- `-i` : 이더넷 카드별 정상/ 에러/ 드랍 송수신 패킷 수 확인
- `-r` : 라우팅 테이블
- `-s` : 네트워크 통계
