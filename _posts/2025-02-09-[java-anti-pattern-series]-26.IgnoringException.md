# Ignoring Exception

- 예외를 무시하면 예측할 수 없는 동작, 버그 및 보안 취약점이 발생할 수 있으므로 예외를 무시하는 것은 나쁜 습관으로 간주된다.

## 이유
1. 정보 손실
2. 정의 되지 않은 동작
3. 리소스 유출
4. 사용자 경험 저하

## 예시
1. catch block에서 아무 것도 하지 않는 경우
2. 아예 예외 처리를 안 한 경우

## 처리 방안
1. 예외를 최대한 작은 범위로 특정해서 지정
2. Try-Catch-Finally 사용
3. Slf4j를 사용해서 로깅
4. 예외를 다른 예외로 변환해서 던지기
